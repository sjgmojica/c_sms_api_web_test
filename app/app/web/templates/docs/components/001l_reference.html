Definition of Terms

Term Description
Delivery Notification

Report sent by Chikka SMS API to inform you whether the message is successfully delivered or not.
Incoming Messages

Messages sent from a mobile phone/user.
Message Receiver

Web application designed to accept and receive HTTP POST requests (messages) FROM the Chikka SMS API.
Message Sender

Web application designed to send HTTP POST requests (messages) TO the Chikka SMS API.
Notification Receiver

Web application designed to accept and receive HTTP POST requests (delivery notifications) FROM the Chikka SMS API.
Outgoing Messages

Messages you send/reply to mobile phones/users.
Shortcode

Number you have provided during registration. This is where users send the messages you will receive.
SMS

Text message.
