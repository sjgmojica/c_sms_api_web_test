How to Digitally Sign a Request Body

  1. Make sure that a Public Key and Private Key have already been generated. (See Generation of Public and Private Keys)

  2. Prepare the plain-text HTTP POST Request body to be signed.

    Sample HTTP POST Request body:

    URI=/api/request.php&USERNAME=METROBANK2929&PASSWORD=ABC123&MESSAGE_TYPE=PULL&SUB_TYPE=START&MESSAGE_ID=MSG00000000000000000000000000001&TRANSID=01FB96AC1234D54ERYRYTR7YRTUGFJHNVHRYRHUGJNFY546576IHJGHFY576HJFTG4565UJRFT65476TUJGH65867YI687IKHGJNRTY658UYJNMTU6IKGYHJK67I9YI7&HEADER=0B00030A020205045B5000&MSISDN=639211234567&SUFFIX=21&ENCODING=SPECIALBIN&BODY=Welcome+to+METROBANK%21&SERVICE=METROBANK&TEST=0

  3. Using the generated Private Key, sign the string above with SHA512.

    Sample Commands for signing a string with SHA512:

    openssl dgst -sha512 \
            -sign my_private_key.pem \
            -out messagefile.txt.sha512 \
            messagefile.txt

  4. Include the digitally signed HTTP POST body as a parameter to the HTTP POST Header for the request.
